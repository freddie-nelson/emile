## Getting Started ðŸš€

Navigate into the project directory:

```bash
cd emile # Or whatever you named the project
```

Get set up quickly using the emile CLI:

```bash
# On Windows
./emile.cmd setup

# On MacOS/Linux
./emile setup
```

To start the development server:

```bash
# On Windows
./emile.cmd dev server
./emile.cmd dev client

# On MacOS/Linux
./emile dev server
./emile dev client
```
